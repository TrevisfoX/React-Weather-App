# Weather application

Own project. Web application that allows you to see the weather in the selected city at the moment, for the whole week, as well as for 21 forward from the current time.

## API

    https://openweathermap.org/

## Technology

    - JavaScript
    - React.js
    - Redux
    - HTML
    - SCSS
